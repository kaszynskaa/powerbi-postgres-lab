3. Database Layer

Schema

A single denormalised table is used, appropriate for a small-scale analytics lab. It stores one row per sale transaction.

CREATE TABLE sales (
    id           SERIAL PRIMARY KEY,
    product_name VARCHAR(100),
    category     VARCHAR(50),
    sale_date    DATE,
    quantity     INTEGER,
    unit_price   NUMERIC(10,2)
);

4. DAX Measures

One measure is defined in dax/measures.dax. It iterates the public sales table and computes revenue row by row using SUMX, which correctly handles row-level arithmetic before aggregation.

Revenue =
SUMX(
    'public sales',
    'public sales'[quantity] * 'public sales'[unit_price]
)
Why SUMX over SUM? — SUM(quantity) * SUM(unit_price) would give an incorrect result when multiple products are present. SUMX multiplies per row first, then aggregates.

7. CI / Quality Gates

A GitHub Actions workflow (.github/workflows/sql-lint.yml) runs sqlfluff against all files in sql/ on every push and pull request to master.

sqlfluff lint sql/ --dialect postgres

Configuration is managed via .sqlfluff:

  • Dialect set to postgres
  • Keywords enforced as UPPER case
  • Identifiers enforced as lower case
  • Wildcard SELECT rule (AM04) excluded for analytical queries

8. Key Findings

  • Revenue concentration — Laptop alone generates 88.9% of total revenue (90,000 PLN of 101,250 PLN).
  • Category split — Electronics: 94.8% vs Accessories: 5.2%.
  • Volume vs value — Accessories have higher unit counts but far lower revenue per sale.
  • Trend — Revenue peaks on Day 1 (Laptop) and drops sharply — suggests a need for more diverse high-value products.

9. Next Steps

  1. Add a date dimension table for time intelligence DAX functions (YTD, MoM).
  2. Introduce a products dimension to normalise the schema.
  3. Add more DAX measures: Average Order Value, Units Sold YTD, Revenue Growth %.
  4. Publish the report to Power BI Service for sharing and scheduled refresh.
  5. Expand CI to include a database integration test that seeds and queries a temporary PostgreSQL container.
